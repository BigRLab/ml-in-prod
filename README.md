# ml-in-prod
A python predictive system design.

## Building the pipeline
- `cd training`
- `virtualenv venv`
- `source venv/bin/activate`
- `pip install -r requirements.txt`
- `python training.py`


## Building the pipeline
- If you did the previous steps then: `cd ../; deactivate`
- `cd server`
- `virtualenv venv`
- `source venv/bin/activate`
- `pip install -r requirements.txt`
- `python main.py`
